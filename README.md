# Amo_LIMS
Laboratory Information Management System for Amogene.

前期需求共分三个模块：
1. ### 管理员模块
- 管理员登录
- 管理员权限
- 管理员个人信息

2. ### 客户模块
- 客户信息档案
- 客户邮件发送
- 客户信息填写

3. ### 数据整理模块
- 数据上传
- 数据归档
- 数据检索


## 管理员模块
- 以OpenID登入，不设置注册系统；
- 设置管理员权限，用于创建项目号，修改项目内容；
- 管理员信息字段固定，数据表如下:
|字段    |属性    |
|:------|:------|
|占位1   |        |
|占位2   |        |
|占位3   |        |

## 客户模块
- 邮件系统样式固定，主要发送"建库测序表"和"数据分析表"；
- 客户信息字段固定，数据表如下:
|字段    |属性    |
|:------|:------|
|占位1   |        |
|占位2   |        |
|占位3   |        |

## 数据整理模块
- 外部用户填写：以管理员创建项目号后，生成唯一验证码，用于外部用户填写时的校验；

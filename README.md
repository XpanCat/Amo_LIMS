# Amo_LIMS
Laboratory Information Management System for Amogene.

前期需求共分三个模块：
1. ### 管理员模块
1.1 管理员登录
1.2 管理员权限
1.2 管理员个人信息

2. ### 客户模块
2.1 客户信息档案
2.2 客户邮件发送
2.3 客户信息填写

3. ### 数据表整理模块
3.1 数据上传
3.2 数据归档
3.3 数据检索


## 管理员模块
- 以OpenID登入，不设置注册系统；
- 设置管理员权限，用于创建项目号，修改项目内容；
- 管理员信息数据表
